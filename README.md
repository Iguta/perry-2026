# perry-2026

A modern, dark-themed personal goal and task tracker designed to boost productivity with clear visuals and smooth interactions.

## Overview
perry-2026 helps individuals organize tasks, set goals, and visualize progress across daily, weekly, monthly, and yearly views. The app focuses on clarity and responsiveness without requiring authentication.

## Key Features
- **Task Management**: Add, edit, delete, and view tasks with description, due date, and priority (low/medium/high).
- **Goal Management**: Set New Year resolutions, monthly goals, and daily goals with themed categories (Spiritual, Physical Fitness, Academics, Career) plus custom categories.
- **Progress Tracking**: Dashboard charts showing progress for today, this week, this month, and this year using percentage/hybrid metrics.
- **Calendar View**: Visualize tasks by date with interactive day expansion to drill into scheduled activities.

## UI/UX
- Modern, minimalistic interface
- Default dark theme with soft accent colors
- Responsive across mobile, tablet, and desktop
- Smooth interactions and subtle animations
